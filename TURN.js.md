### APi
```javasciript
book.turn('page') // 当前页码
book.turn('page', 2) // 跳转到指定页
book.turn('stop') // 停止当前页的过渡
book.turn('pages') // 总页码
book.turn('center') // 书本居中显示
book.turn('peel', 'br') // 右下角显示翻角
```

### Corners
  - tl: Top left.
  - tr: Top right.
  - r: Right edge (hard pages only)
  - br: Bottom right.
  - bl: Bottom left.
  - l: Left edge (hard pages only)
